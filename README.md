# AP Biology Infinite FRQ Generator

Generate unlimited, College Board-aligned AP Biology Free Response Questions, answer them on paper, upload a scan, and get per-part AI grading + feedback.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
